1. Kiểm tra Javascript cơ bản

Bài 1: Tìm chuỗi có độ dài lớn nhất

Viết function truyền vào mảng các chuỗi có độ dài bất kỳ. Kết quả trả về là 1 mảng các chuỗi có độ dài lớn nhất.

Ví dụ: getStringsWithMaxLength(['aba', 'aa', 'ad', 'c', 'vcd']) sẽ trả về ['aba', 'vcd'].

Bài 2: Xử lý mảng users

Cho mảng users như sau:

users = [
      {
          name: "Bùi Công Sơn",
          age: 30,
          isStatus: true
      },
      {
          name: "Nguyễn Thu Hằng",
          age: 27,
          isStatus: false
      },
      {
          name: "Phạm Văn Dũng",
          age: 20,
          isStatus: false
      }
  ]

Yêu cầu:

  1. Viết function truyền vào 1 mảng các object user. Trả về mảng các user có age > 25 và isStatus = true
  2. Viết function truyền vào 1 mảng các object user. Trả về mảng các user có age tăng dần

Bài 3: Đếm số lần xuất hiện của các phần tử trong mảng

Viết function truyền vào 1 mảng các chuỗi. Trả về Object hiển thị xem mỗi phần tử trong mảng xuất hiện bao nhiêu lần.

Ví dụ:

getCountElement(["one", "two", "three", "one", "one", "three"])
{
      one: 3,
      two: 1,
      three: 2
  }
